美团面经

部门：

交易结算-上海-资金基础能力（已oc）

时间线：

8.28 投递简历 -> 8.30 一面 -> 8.30 面试后45min商量offer -> 9.2 正式offer

面试：

  1. 自我介绍

    1. 做个简单的自我介绍
    2. 能实习多久、最快多久能到

  2. 项目相关

    1. 做过哪些项目、简单介绍下简历上的两个、是单体还是微服务、项目中负责了哪些部分，都有亲自参与吗、项目有没有实际部署
    2. 项目中秒杀业务用到了分布式锁和lua脚本，具体怎么用的，lua脚本中写了哪些内容，为什么这么用（举个case），为什么要保证原子性，为什么用Redission，与直接用Setnx的区别。
    3. 项目中用到了mq，为什么引入mq，具体怎么用的，mq还有哪些用处？
    4. 项目中还用到了redis哪些数据结构，具体怎么用的？
    5. 简单介绍下第二个项目（微服务相关，应该是不感兴趣）

  3. 简历相关

    1. Java基础
      1. 常用的Java集合有哪些，讲一下，底层实现（包括插入过程等），是否线程安全，哪些线程安全，实现原理
      2. Java异常了解不了解，谈一下。
    2. JVM
      1. 讲一下JVM的内存结构，哪些线程共享？哪些不共享？
      2. minorgc和full gc的区别？
      3. 出现OOM了怎么排查？（Jmap，Jstack）是否看过dump文件？
    3. JUC
      1. 乐观锁、悲观锁
      2. CAS怎么实现
      3. volatile有什么用
      4. synchronized的锁升级具体过程
      5. 创建线程池需要指定的参数
    4. Spring
      1. 了解不了解dispatch serverlet
      2. mybatis有哪些常用标签
      3. #{} 与 ${}的区别
      4. spring有哪些常用标签
      5. auto wired和 resource的区别，名称注入是什么名称（类名_小写（还是驼峰？忘记了））
    5. Mysql
      1. 了解不了解索引，怎么设计索引
      2. 为什么用B+树做索引结构
      3. 什么情况下索引失效
      4. 最左匹配的时候，设计索引有哪些参考原则
      5. 有没有过慢sql调优经历
      6. explain字段都了解不了解，以及关于字段的一系列引申问题
      7. 手写一个sql（查询不及格学生？好像是，主要忘了还有having这个用法）
    6. 设计模式
      1. 了解哪些设计模式
      2. 讲讲策略模式
      3. 手写一个线程安全的单例模式

  4. 手撕环节

    1. 三数之和

      题目：三数之和

      比较常规，不到10min秒了

  5. 反问环节

    1. 部门业务主要做什么，氛围如何
    2. 听业务介绍，部门比较核心，比较担心进去后是否打杂，能否有很大成长
